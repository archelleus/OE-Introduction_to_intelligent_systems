# Fuzzy Logic and ANFIS Project

## Q1: Washing Machine Fuzzy System
- Inputs: Dirt Level, Load Size
- Output: Cycle Time
- Method: Mamdani FIS
- Membership functions: Triangular
- Rules: 7 rules defined manually

## Q2: Student Performance Prediction
- Inputs: Attendance, Assignment, Test
- Output: Performance Level
- Method: ANFIS (Neuro-Fuzzy System)
- Model trained using dataset

## Tools Used
